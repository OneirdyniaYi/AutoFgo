# AutoFgo

- Run `Fgo.grab(fgo.area['xxx'], fname='xxx_sample.jpg')` to get image files.
- Replace the screenshot in `./data/menu_sample.jpg` with your own images for running.
- Edit `config.py` if you want.
- Run `main.py` and have fun!

### Requirements

- **On Unix/Linux:**
  - pillow 
  - pyuserinput
  - numpy 
  - autopy
- **Windows**
  - pywin32
  - pillow
  - numpy
